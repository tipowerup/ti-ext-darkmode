# Dark mode Extension

A custom TastyIgniter extension

## Installation

```bash
composer require tipowerup/ti-ext-darkmode
```

Then install in TastyIgniter:

```bash
php artisan igniter:extension-install tipowerup.darkmode
```

## Requirements

- TastyIgniter v4.0+
- PHP 8.2+
